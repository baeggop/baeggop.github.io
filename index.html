Store API

Store API provides endpoints to manage store-related data, including fetching store details, managing tables, and interacting with menus.

  • 1. GET /getAll - 모든 가게 조회
  • 2. GET /search - 가게 검색
  • 3. GET /:storeId/tables - 특정 가게의 테이블 정보 조회
  • 4. GET /:storeId/orders - 특정 가게의 미결제 주문 조회
  • 5. GET /:tableId/orders - 특정 테이블의 미결제 주문 조회
  • 6. GET /:storeId/menus - 특정 가게의 메뉴 조회
  • 7. GET /:storeId - 특정 가게의 상세 정보 조회
  • 8. POST /:storeId/tables - 테이블 정보 수정 및 등록
  • 9. POST /:storeId/menu-categories - 메뉴 카테고리 추가
  • 10. POST /:menuCateId/menus - 메뉴 추가
  • 11. POST /:menuId/menu-options - 메뉴 옵션 추가

Order API

Order API provides endpoints to manage order-related functionalities, such as fetching unpaid orders and creating new orders for tables.

  • GET /:tableId - Fetch unpaid orders for a specific table
  • POST /:tableId - Create or update an order for a table

Payment API

Payment API provides endpoints to update the payment status of orders, ensuring seamless transaction management.

  • PUT /:orderId - Update the payment status of an order
