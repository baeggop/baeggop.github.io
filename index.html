Store API

Store API는 가게 세부 정보를 가져오거나, 테이블을 관리하며, 메뉴와 상호작용할 수 있는 엔드포인트를 제공합니다.

  • GET /getAll - 모든 가게 조회
  • GET /search - 이름 또는 주소로 가게 검색
  • GET /:storeId/tables - 특정 가게의 테이블 및 미결제 주문 조회
  • GET /:storeId/orders - 특정 가게의 미결제 주문 조회
  • GET /:tableId/orders - 특정 테이블의 미결제 주문 조회
  • GET /:storeId/menus - 특정 가게의 메뉴 및 메뉴 옵션 조회
  • GET /:storeId - 특정 가게의 세부 정보 및 주문 정보 조회
  • POST /:storeId/tables - 가게의 테이블 생성 및 수정
  • POST /:storeId/menu-categories - 가게에 새로운 메뉴 카테고리 추가
  • POST /:menuCateId/menus - 메뉴 카테고리에 새로운 메뉴 추가
  • POST /:menuId/menu-options - 메뉴에 새로운 옵션 추가

Order API

Order API는 테이블 주문을 관리하며, 특정 테이블에 대한 주문 생성 및 미결제 주문을 조회할 수 있습니다.

  • GET /:tableId - 특정 테이블의 미결제 주문 조회
  • POST /:tableId - 테이블에 새로운 주문 생성

Payment API

Payment API는 주문의 결제 상태를 업데이트할 수 있습니다.

  • PUT /:orderId - 주문 결제 상태 업데이트
