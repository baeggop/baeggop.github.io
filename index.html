Store API

Store API provides endpoints to manage store-related data, including fetching store details, managing tables, and interacting with menus.

  • GET /getAll - Fetch all stores
  • GET /search - Search stores by name or address
  • GET /:storeId/tables - Fetch tables and unpaid orders for a specific store
  • GET /:storeId/orders - Fetch unpaid orders for a store
  • GET /:tableId/orders - Fetch unpaid orders for a specific table
  • GET /:storeId/menus - Fetch menu and menu options for a store
  • GET /:storeId - Fetch store details and order information
  • POST /:storeId/tables - Create or update tables for a store
  • POST /:storeId/menu-categories - Add a new menu category to a store
  • POST /:menuCateId/menus - Add a new menu to a menu category
  • POST /:menuId/menu-options - Add a new menu option to a menu

Order API

Order API manages table orders, including creating orders and fetching unpaid orders for specific tables.

  • GET /:tableId - Fetch unpaid orders for a specific table
  • POST /:tableId - Create a new order for a table

Payment API

Payment API updates the payment status of orders.

  • PUT /:orderId - Update payment status for an order
