목차

  • Base URL: /payment
  • 1. PUT /:orderId - 주문 결제 상태 업데이트
설명: 특정 주문의 결제 상태를 "Y"로 업데이트합니다.
Request
orderId: 주문 ID (string, 필수)
Response
{
  "message": "Order payment updated successfully",
  "order": {
    "id": "string",
    "isPaid": "Y"
  }
}
Error Responses
{
  "error": "Order not found"
}
{
  "error": "Failed to update order payment status"
}
