목차

  • Base URL: /store
  • 1. GET /getAll - 모든 가게 조회
  • 2. GET /search - 가게 검색
  • 3. GET /:storeId/tables - 특정 가게의 테이블 정보 조회
  • 4. GET /:storeId/menus - 특정 가게의 메뉴 조회
  • 5. GET /:storeId - 특정 가게의 상세 정보 조회
  • 6. POST /:storeId/tables - 테이블 정보 수정 및 등록
  • 7. POST /:storeId/menu-categories - 메뉴 카테고리 추가
  • 8. POST /:menuCateId/menus - 메뉴 추가
  • 9. POST /:menuId/menu-options - 메뉴 옵션 추가
